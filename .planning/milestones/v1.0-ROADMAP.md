# Roadmap: obFileType

## Overview

Deliver a simple Obsidian plugin that allows opening and editing text-based files like JSON and YAML, with syntax highlighting and configurable file extensions.

## Phases

- [ ] **Phase 1: File Handling** - Users can view and edit arbitrary text files inside Obsidian
- [ ] **Phase 2: Syntax Highlighting** - Render text formats with appropriate coloring
- [ ] **Phase 3: Settings** - Let users configure supported extensions

## Phase Details

### Phase 1: File Handling
**Goal**: Users can view and edit arbitrary text files inside Obsidian
**Mode**: mvp
**Depends on**: Nothing (first phase)
**Requirements**: [FILE-01, FILE-02, FILE-03, FILE-04]
**Success Criteria** (what must be TRUE):
  1. A user can open a .json file inside Obsidian, modify its contents and save it.
  2. Changes made inside Obsidian are correctly persisted to the disk format without data corruption.
