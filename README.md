# obFileType

Open any text file natively in Obsidian.

obFileType is a native file editor for Obsidian that allows you to easily open text files and handle arbitrary text extensions within your workspace. It acts as a lightweight plaintext editor built right into your favorite note-taking app.

![Demo](demo.gif) <!-- USER: Please replace demo.gif with actual animated GIF screen recording -->

## Features
- **Native Experience:** Operates as a native file editor inside Obsidian.
- **Universal Text Support:** Effortlessly open text files with arbitrary text extensions.
- **Plaintext Editing:** A fully-featured plaintext editor with syntax highlighting for various formats like JSON, YAML, and more.
- **Seamless Integration:** Works without launching external editors, keeping you in the flow.

## Supported File Types
- `.json`
- `.yaml` / `.yml`
- `.xml`
- `.csv`
- `.c`, `.cpp`, `.h`, `.hpp`
- Any other plaintext format you configure!

## Installation
Currently, obFileType can be installed manually.

### Manual Installation
1. Go to the Releases page.
2. Download `main.js`, `manifest.json`, and `styles.css`.
3. Create a folder named `obFileType` in your vault's `.obsidian/plugins/` directory.
4. Place the downloaded files into the `obFileType` folder.
5. Reload Obsidian and enable the plugin in Community Plugins settings.
